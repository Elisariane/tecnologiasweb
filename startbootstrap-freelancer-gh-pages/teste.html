Python básico

Nessa categoria você verá o básico de python para poder começar seus projetos e aprofundar seus conhecimentos nessa maravilhosa linguagem.


O que é Python? Python é uma linguagem de programação de alto nível e interpretada que foi criada por Guido Van Rossum em 1989. Foi lançada em 1991, o que resulta em uma excelente linguagem de propósito geral capaz de criar qualquer coisa desde software de desktop até aplicações web e frameworks.
Para aqueles que estão familiarizados com Java ou C ++, o Python quebrará o molde que você criou para uma linguagem de programação típica. Prepare-se para se apaixonar, com o Python!

Por que aprender programação em Python?

Python é uma linguagem de programação dinâmica de alto nível. É muito fácil de aprender e fornece digitação poderosa. O código Python tem um estilo muito "natural", pois é fácil de ler e entender (graças à falta de ponto e vírgula e chaves). A linguagem de programação Python é executada em qualquer plataforma, desde o Windows ao Linux, Macintosh, Solaris, etc. A simplicidade do Python é o que torna tão popular. A seguir, um destaque de sua estética:

  • Linguagem altamente legível
  • Visual limpo
  • Menos sintaxe
  • Manipulação de strings
  • Digitação elegante e dinâmica
  • Interpretada
  • Ideal para scripts e aplicação rápida
  • Apto para muitas plataformas

Espere! Python pode fazer mais.

É uma linguagem muito popular em vários domínios, como automação, Big Data, AI, etc. Você pode consultar todo este blog sobre as 10 principais razões para aprender python. Você também ficará impressionado com a grande quantidade de empresas em todo o mundo.

Controle de fluxo

O Controle de fluxo nos permite definir um fluxo na execução de nossos programas. Para imitar o mundo real, você precisa transformar as situações do mundo real em seu programa. Para isso, você precisa controlar a execução de suas declarações de programa usando os fluxos de controle.
Existem seis controles básicos de fluxo usados ​​na programação em Python:
  1. if
  2. for
  3. while
  4. break
  5. continue
  6. pass

If
A declaração composta do Python 'if' permite que você execute blocos de instruções condicionalmente.
Sintaxe da instrução If: 
if expressão:
     declaração

elif expressão:
     declaração

elif expressão:
     declaração
...
else:
     declaração




A imagem acima explica o uso da declaração "if" usando um exemplo de login do Facebook.
A página de login do Facebook direcionará você para duas páginas com base no fato de seu nome de usuário e sua senha corresponderem à sua conta.
Se a senha digitada estiver errada, ela direcionará você para a página à esquerda.
Se a senha digitada estiver correta, você será direcionado para sua página inicial.
Vamos agora ver como o Facebook usaria a declaração If: 
senha = facebook_hash(input_senha)
if senha == hash_senha:
   print('Login feito com sucesso!')
else:
   print('Login falhou. Senha incorreta.')

O código acima apenas fornece uma implementação de alto nível da instrução If no exemplo de login do Facebook usado. A função Facebook_hash () usa o input_senha como um parâmetro e compara-o com o valor de hash armazenado para esse usuário específico.

For

A instrução for suporta a execução repetida de uma instrução ou bloco de instruções controladas por uma expressão iterável.
Sintaxe da instrução For: 
for variável in iterável:
     declaração



A declaração "for" pode ser entendida a partir do exemplo acima.
A lista "Amigos" do seu perfil exibirá os nomes e as fotos de todos os seus amigos;
Para conseguir isso, o Facebook recebe sua lista de "lista de amigos" contendo todos os perfis de seus amigos;
O Facebook começa a exibir o HTML de todos os perfis até o índice da lista atingir "NULL" A ação de preencher todos os perfis na sua página é controlada pela declaração "for".
Vamos agora examinar um programa de amostra no Python para demonstrar a instrução For. 
viajando = input("Você esta viajando? Sim ou Não:")
while viajando == 'sim':

print("Por favor volte de novo.")


While

A instrução while na programação Python suporta execução repetida de uma instrução ou bloco de instruções que é controlada por uma expressão condicional.
Sintaxe da instrução While: 
while expressão:
     declaração


Usaremos o feed de notícias do Facebook acima para entender o uso do loop while.
Quando entramos em nossa homepage no Facebook, temos cerca de 10 histórias carregadas em nosso newsfeed;
Assim que chegamos ao final da página, o Facebook carrega mais 10 histórias em nosso newsfeed;
Isso demonstra como o loop "while" pode ser usado para atingir isso.
Vamos agora examinar um exemplo de programa em Python para demonstrar a instrução While. 
count = 0
print('Mostrando números de 0 a 9)
while (count < 10):
   print('O count é ',count)
   count = count+1
print('Fim')


Este programa imprime números de 0 a 9 usando a instrução while para restringir o loop até atingir 9.

Break

A instrução break é permitida somente dentro de um corpo de loop. Quando o break é executado, o loop termina. Se um loop estiver aninhado dentro de outros loops, o break terminará apenas o loop aninhado mais interno.
Sintaxe da declaração Break: 
while True:
     x = get_next()
     y = preprocess(x)
     if not keep_looking(x, y): break
     process(x, y)



A declaração de controle de fluxo "break" pode ser entendida a partir do exemplo acima. Vamos considerar o caso de um alarme em um toque móvel em um determinado momento.
Suponha que o telefone receba uma chamada no momento em que o alarme está tocando, o alarme é interrompido imediatamente e a campainha do telefone começa a tocar.
É assim que o break funciona essencialmente.
Vamos agora ver um exemplo de programa em Python para demonstrar a declaração Break. 
for letter in 'The Quick Brown Fox. Jumps, Over The Lazy Dog':
   if letter == '.':
      break
   print ('Current Letter :', letter)


Este programa imprime todas as letras em uma determinada string. Ele quebra sempre que encontra um "." Ou um ponto final. Nós fizemos isso usando a instrução Break.
Continue

A instrução continue é permitida somente dentro de um corpo de loop. Quando continue executa, a iteração atual do corpo do loop termina e a execução continua com a próxima iteração do loop.
Sintaxe da instrução Continue: 
for x in some_container:
    if not seems_ok(x): continue
    lowbound, highbound = bounds_to_test()
    if x<lowbound or x>=highbound: continue
    if final_check(x):
        do_processing(x)




Exemplo: A instrução Continue pode ser entendida usando a chamada recebida e um alarme.
Suponha que estamos em uma chamada e o alarme está agendado durante o tempo de chamada, então o gatilho de alarme reconhece o evento de chamada
Uma vez que o evento de chamada é anotado, o telefone continua o alarme para tocar no próximo período de soneca

Vamos agora ver um exemplo de programa em Python para demonstrar a instrução Continue. 
for num in range(10, 21):
   if num % 5 == 0:
      print ("Encontrada a multiplicação de 5")
      pass
      num = num + 1
      continue
   print ("Encontrado o número: ", num)

Este programa imprime todos os números, exceto os múltiplos de 5, de 10 a 20.


Pass
A instrução "pass", que não executa nenhuma ação, pode ser usada como um espaço reservado quando uma instrução é sintaticamente necessária, mas você não tem nada específico a fazer.

Sintaxe da instrução Pass: 
if condição1(x):
    processo1(x)
elif x>23 or condição2(x) and x<5:
    pass
elif condição3(x):
    processo3(x)
else:
    process_default(x)


Agora, vamos examinar um programa de amostra no Python para demonstrar a instrução Pass. 
for num in range(10, 21):
    if num % 5 == 0:
       print ("Encontrado múltiplo de 5: ")
       pass
       num++
    print ("Encontrado número: ", num)

Este programa imprime os múltiplos de 5 com uma sentença separada.

Manipulação de arquivos


Manipulação de Arquivo refere-se àquelas operações que são usadas para ler ou gravar um arquivo.
Para realizar o manuseio de arquivos, precisamos executar estas etapas:
  1. Abrir arquivo
  2. Arquivo de leitura / gravação
  3. Fechar arquivo




Abrindo um arquivo
  • Python tem uma função interna open () para abrir um arquivo
  • Esta função retorna um objeto de arquivo, também chamado de handle, como é usado para ler ou modificar o arquivo de acordo

Exemplo de programa: 
file = open("C:/Users/Zombie Code/Hello.txt", "r")
for line in file:
   print (line)


Escrevendo para um arquivo
  • Para escrever em um arquivo, precisamos abri-lo em modo de gravação "w", acrescentar "a" ou criação exclusiva "x"
  • Precisamos ter cuidado com o modo "w", pois ele será substituído no arquivo, se já existir. Todos os dados anteriores são apagados
  • Escrever uma string ou seqüência de bytes (para arquivos binários) é feito usando o método write ()


with open("C:/Users/Zombie Code/Writing_Into_File.txt", "w") as f
f.write("First Line\n")
f.write("Second Line\n")

file = open("D:/Writing_Into_File.txt", "r")
for line in file:
   print (line)


Lendo De Um Arquivo
  • Para ler o conteúdo de um arquivo, devemos abrir o arquivo no modo de leitura
  • Podemos usar o método de leitura (tamanho) para ler o tamanho do número de dados
  • Se o parâmetro de tamanho não for especificado, ele lê e retorna até o final do arquivo

file = open("C:/Users/Zombie Code/Writing_Into_File.txt", "r")
print(file.read(5))
print(file.read(4))
print(file.read())


Fechando um arquivo
  • Quando terminamos as operações no arquivo, precisamos fechá-lo corretamente.
  • Fechar um arquivo libera os recursos que foram amarrados com o arquivo e é feito usando o método close ().

Exemplo de programa: 
file = open("C:/Users/Zombie Code/Hello.txt", "r")
text = file.readlines()
print(text)
file.close()


Objeto e Classe



Python é uma linguagem de programação orientada a objetos. O objeto é simplesmente uma coleção de dados (variáveis) e métodos (funções) que atuam nesses dados. Class é um modelo para o objeto.


Definindo Uma Classe
Nós definimos uma classe usando a palavra-chave "Class". A primeira string é chamada docstring e tem uma breve descrição sobre a classe. 

class MyNewClass:
'''Isto é um docstring. Eu criei uma nova classe '''
pass


Criando um Objeto
Um objeto de classe pode ser usado para criar novas instâncias de objeto (instanciação) dessa classe. O procedimento para criar um objeto é semelhante a uma chamada de função. 
ob = MyNewClass
