Jquery Panorama Viwer: Tutorial



O que é?

É um plugin jquery com utilidade para fotos panorâmicas, de maneira bem simples mantem a largura normal das fotos panorâmicas e permite que o usuário interaja "rolando" pela imagem.


Compatibilidade: tanto em computadores quanto smartphones. Não existe compatibilidade com o Internet Explorer.


Como o plugin funciona?
Em vez de ter um contêiner div com a imagem que se move quando você arrasta, é usado a abordagem de plano de fundo do CSS. A razão por trás disso é que ele permite que o usuário controle a altura da foto panorâmica sem precisar defini-la explicitamente na função, ao mesmo tempo mantendo a capacidade de resposta.

Veja como isso funciona:

Primeiro, o Panoramic Viewer irá envolver todas as fotos com um contêiner de invólucro para que ele possa conter toda a largura da foto sem expandir o layout. Feito isso, o plug-in criará outro contêiner, atribuirá a foto como plano de fundo desse contêiner usando o estilo de plano de fundo e tamanho de plano de fundo (ou responsivo) do CSS e detectará uma ação de arrastar.
Ao arrastar, em vez de mover o contêiner, o plug-in moverá a posição de plano de fundo e, como tudo está sendo movido por CSS, podemos controlar o tempo de animação e o comportamento de suavização sem tocar na parte JS.





E é assim que o plug-in funciona. Agora, você terá uma foto panorâmica interativa incorporada em seu site.

$(".panorama").panorama_viewer({
    repeat: false,
    direction: "horizontal",
    animationTime: 700,
    easing: "ease-out",
    overlay: true
  });

 
 


repeat: se você tem uma foto panorâmica 360 perfeita e deseja que o usuário possa girar sem
estar limitado pelas dimensões da foto, defina-a como true e o fundo se repetirá para sempre.
O valor padrão é false (falso). 

direction: caso você tenha uma impressionante foto panorâmica 360 vertical,
você pode definir essa opção como vertical para que o espectador possa rolar para cima em vez
de para o lado. O valor padrão é horizontal. 

animationTime: Você pode definir quanto tempo a animação de atenuação irá durar com essa opção.
A opção aceita valores em milissegundos (sem a unidade ms).
Mude para 0 se quiser que o arrasto seja instantâneo. O valor padrão é 700. 

easing: Você pode definir sua opção de atenuação aqui.
Esta opção aceita a função usual de temporização de transição do CSS3.
As opções disponíveis são “ease” (“facilidade”), “linear” (“linear”) ,“ease-in” (“facilidade de entrada”), “ease-out” ( “facilidade de saída”), “ease-in-out” , e  “cubic-bezier(…))”.
O valor padrão é “ease-out”. 

overlay: para tornar a interação clara para o usuário,
foi adicionada uma sobreposição para indicar que esta é uma foto panorâmica.
Se você quiser desativar isso, alterne essa opção para false.
O valor padrão é true.
